# Operations Checklists

This directory contains operational checklists for common NovaSight procedures.

## Available Checklists

| Checklist | Purpose | Use When |
|-----------|---------|----------|
| [Pre-Deployment](pre-deployment.md) | Verify readiness before deployment | Before any deployment |
| [Production Release](production-release.md) | Step-by-step production release | Deploying to production |
| [Incident Response](incident-response.md) | Handle production incidents | During incidents |
| [Maintenance Window](maintenance-window.md) | Scheduled maintenance | Planned maintenance |
| [Security Incident](security-incident.md) | Security breach response | Security events |

## How to Use

1. Copy the relevant checklist to a new document or issue
2. Complete each item in order
3. Mark items as done when completed
4. Document any deviations or issues
5. Archive completed checklists for audit purposes

## Checklist Updates

These checklists should be reviewed and updated:

- After any incident where the checklist was insufficient
- Quarterly as part of operational review
- When new infrastructure or procedures are added
