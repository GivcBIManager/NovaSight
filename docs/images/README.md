# NovaSight Documentation Images

This directory contains images used in the NovaSight documentation.

## Image Guidelines

### File Naming

Use descriptive, kebab-case names:
- `connect-datasource.png`
- `dashboard-builder.png`
- `semantic-layer-model.png`

### File Formats

| Format | Use Case |
|--------|----------|
| PNG | Screenshots, diagrams with transparency |
| JPG | Photos, large images |
| SVG | Icons, logos, simple graphics |
| GIF | Short animations, interactions |

### Image Sizes

| Type | Max Width | Recommended |
|------|-----------|-------------|
| Full-width | 1200px | 1200px |
| In-line | 800px | 800px |
| Thumbnails | 400px | 400px |

### Compression

Compress images before adding:
- Use tools like TinyPNG, ImageOptim
- Target file size under 200KB
- Balance quality vs. load time

## Creating Screenshots

### Guidelines
1. Use a clean browser window (no bookmarks bar, extensions)
2. Use consistent window size (1920x1080 recommended)
3. Use light theme for screenshots
4. Highlight relevant areas with annotations if needed
5. Remove any sensitive or test data

### Annotation Style
- Use blue (#3B82F6) for highlights
- Use red (#EF4444) for warnings/errors
- Add numbered callouts for step-by-step guides
- Keep annotations minimal and clean
