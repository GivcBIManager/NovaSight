# Keyboard Shortcuts

A comprehensive list of keyboard shortcuts available in NovaSight.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` | Open command palette |
| `Ctrl + /` | Open keyboard shortcuts help |
| `Ctrl + S` | Save current item |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |
| `Escape` | Close modal / Cancel |
| `?` | Show contextual help |

## Customizing Shortcuts

To customize keyboard shortcuts:

1. Go to **Settings** > **Keyboard Shortcuts**
2. Find the action you want to customize
3. Click the current shortcut
4. Press your desired key combination
5. Click **Save**

### Reset to Defaults

1. Go to **Settings** > **Keyboard Shortcuts**
2. Click **Reset All to Defaults**
3. Confirm

## Platform-Specific Notes

### macOS

Replace `Ctrl` with `Cmd` (⌘) for most shortcuts:
- `Cmd + S` - Save
- `Cmd + Z` - Undo
- `Cmd + Enter` - Execute query

### Linux

Shortcuts work as documented with `Ctrl`.

### Windows

Shortcuts work as documented.
